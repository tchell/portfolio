# Build
- Remove live.js
    + `git rm live.js`
    + remove \<script href="live.js"> from index.html
- send client_id
    + `scp PATH_TO_PROJECT/client_id.json REMOTE_USER@REMOTE_URL:PATH_TO_PROJECT`
